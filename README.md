# BandpassFilter
Swift bandpass filter

## Install

Add `https://github.com/t0rn/BandpassFilter` in the [“Swift Package Manager” tab in Xcode](https://developer.apple.com/documentation/xcode/adding_package_dependencies_to_your_app).
